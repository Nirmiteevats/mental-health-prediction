# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-10

### Added
- Initial release of the Mental Health Prediction machine learning model.
- Streamlit interactive web application (`app.py`).
- Jupyter notebook for model training and EDA.
- Project restructuring for production readiness (`data/`, `models/`, `notebooks/`, `tests/`).
- Automated tests using `pytest`.
- GitHub Actions CI workflow for linting and testing.
- Comprehensive documentation (README, CONTRIBUTING, CODE_OF_CONDUCT, CHANGELOG, LICENSE).
